# Grid Demand Forecasting

This framework provides an advanced forecasting system designed to predict power grid load using real-time data from major ISOs. By integrating live data streams via `gridstatus`, it eliminates the need for static files and ensures predictions reflect current conditions. The underlying `ResidualGridNet` architecture uses a Bi-LSTM network with an attention mechanism to model complex temporal dependencies, delivering accurate predictions across multiple horizons (24h, 48h, and 72h). Optimized for spinning reserve management, the system features a robust training pipeline with a domain-specific weighted loss and is fully encapsulated in an interactive Jupyter Notebook for seamless data fetching and model training.
